== Description ==

= Cryptocurrency Pricing list and Ticker =

Cryptocurrency Pricing list and Ticker Plugin you can use to list all the cryptocurrencies such as bitcoin, litecoin, ethereum, ripple, dash etc. It has more than 2000+ coins that you can choose to show on your website. We are using [Crypto Compare](https://www.cryptocompare.com) API which is having 99.9% uptime for API calls.

= Currency Ticker and List =
We have integrated additional feature to the plugin which you will love to use on your website. Now you can add ticker or list widget to your website with more than 30 physical currency selection. This feature comes with full customization and you can add as many list or ticker on the same page you wanted. There is no limitation.

[See Ticker demo](http://plugins.premium-themes.co/ccpw/currency-ticker/)
[See Ticker PRO demo](https://plugins.premium-themes.co/ccpwpro/ticker/)

<ul>
<li>You can edit the colors of text, background and borders.</li>
<li>Show logos or Cryptocurrencies or Physical currencies.</li>
<li>You can add more than one ticker to the same page.</li>
<li>Price will automatically change to the latest price of that currency.</li>
<li>You can choose the base currency like USD, AUD, CAD, EUR, etc. There are more than 30 currencies we added to choose.</li>
<li>You can show currencies as a flag or just short name.</li>

We are using [CoinMarketcap](https://coinmarketcap.com/) API for this feature.

== Installation ==

1. Download the zip file and save it locally - cryptocurrency-pricing-list.zip

2. Log in to your WordPress blog administration screen.

3. Click on 'Plugins' and then on 'Add New' (left menu)

4. Click on the 'Upload' link, choose 'cryptocurrency-pricing-list.zip' file and click on the 'Install' now button.

5. Click on 'Activate Plugin' link

6. Click on the CCPW tab

7. Select the currencies you would like to show on your web page

8. You can go to Currency Calculator Tab in order to create or get Calculator shortcode. 


== Frequently Asked Questions ==

**Q: Is this plugin free or paid?**

A: This plugin is totally free there are no hidden charges or limitations of this plugin you can customize these according to your needs.


**Q: Can I choose local currency or other physical currencies?**

A: We have added many physical currencies that you can use to show the price comparision in between crypto currencies and others.


**Q: Do I need a credit card to sign up?**

A: You don't need to input or save your credit card. We dont need you to create your account anywhere to use this plugin. Its purely stand-alone plugin.

**Q: What currencies do you support?**

A: We support more than 2000 crypto currencies and also we have added more than 20 Physcial currnecies to use for comparision.
